Project Overview

This project analyzes Superstore sales data using an interactive Tableau dashboard. The objective is to evaluate sales performance, profit margins, and return trends across regions, categories, and customer segments to support data-driven business decisions.

Business Problem

Retail businesses require clear visibility into sales and profitability across locations and product categories. Large datasets make it difficult to quickly identify high-performing regions, loss-making segments, and overall performance trends without effective visualization.

Data Analysis & Preparation

Reviewed and prepared structured Superstore data for analysis
Created calculated fields to derive key performance metrics
Applied filters and parameters for dynamic data exploration

Dashboard Features

KPI indicators for Sales, Profit Margin, and Return Rate
Parameter-driven views to switch between Sales, Profit, and Orders
State-wise sales map for regional performance analysis
Category and sub-category performance insights
Interactive filters and dashboard actions

Tools Used

Tableau Desktop (Public Edition)

Key Insights

Identified high-revenue and low-profit regions
Analyzed category and sub-category performance trends
Observed return patterns impacting overall profitability

Conclusion

The Tableau dashboard provides an interactive and visual approach to analyzing retail sales performance, enabling stakeholders to explore trends, compare metrics, and make informed business decisions.
